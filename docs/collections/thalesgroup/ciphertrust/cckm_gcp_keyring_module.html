thalesgroup.ciphertrust.cckm_gcp_keyring module – CCKM module for GCP KeyRings

Note

This module is part of the thalesgroup.ciphertrust collection (version 1.0.0).

To install it, use: ansible-galaxy collection install thalesgroup.ciphertrust.

To use it in a playbook, specify: thalesgroup.ciphertrust.cckm_gcp_keyring.

New in thalesgroup.ciphertrust 1.0.0

  • Synopsis

  • Parameters

  • Examples

Synopsis

  • This is a Thales CipherTrust Manager module for working with the CipherTrust Manager APIs, more specifically with CCKM for GCP KeyRing

Parameters

Parameter

Comments

list / elements=string

Key Ring parameters.

password

project_id

The project id of the key ring.

Examples

- name: "Create GCP KeyRing"
  thalesgroup.ciphertrust.cckm_gcp_keyring:
    localNode:
        server_ip: "IP/FQDN of CipherTrust Manager"
        server_private_ip: "Private IP in case that is different from above"
        server_port: 5432
        user: "CipherTrust Manager Username"
        password: "CipherTrust Manager Password"
        verify: false
    op_type: create
