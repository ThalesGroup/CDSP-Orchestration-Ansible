Thalesgroup.Ciphertrust

Collection version 1.0.0

  • Description

  • Plugin Index

Description

ThalesGroup CipherTrust Data Security Platform Ansible collection to configure various CipherTrust connectors as well as apply policy driven data protection at the time of application orchestration using Ansible.

Author:

  • ThalesGroup <ss@thalesgroup.com>
